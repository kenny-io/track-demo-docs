# Quill Log API
*quill-log/api*

> API reference for quill-log — createLogger, level methods, child loggers, and formatters.

---

## `createLogger(options?)`

Create a logger.

| Option | Type | Default | Description |
| --- | --- | --- | --- |
| `level` | `"debug" \| "info" \| "warn" \| "error"` | `"info"` | Minimum level to emit. |
| `formatter` | `(record) => void` | `jsonFormatter` | Record sink. |
| `base` | `object` | `{}` | Fields merged into every record. |

## `log.debug/info/warn/error(msg, fields?)`

Emit a record at the given level. Records below the configured minimum level
are dropped silently.

## `log.child(fields)`

Derive a logger whose records include extra base fields on top of the
parent's.

```js
const reqLog = log.child({ requestId: "req_456" });
```

## `log.setLevel(level)`

Change the minimum level at runtime without recreating the logger. Throws `RangeError` on unknown levels.

```js
log.setLevel("debug"); // flip to debug during investigation
```

## `log.level()`

Get the current minimum level.

```js
console.log(log.level()); // "info"
```

## `jsonFormatter(record)`

The default formatter: JSON lines on stdout. Write your own function with the
same signature to send records to a file, a socket, or an aggregator.
