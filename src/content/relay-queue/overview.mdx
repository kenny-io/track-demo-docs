Relay Queue is an in-memory job queue for Node.js. Jobs are async functions;
the queue runs up to `concurrency` of them at once, picks the
highest-priority waiting job first, bounds each attempt with a timeout, and
retries failures with exponential backoff before reporting them.

## Install

```bash
npm install relay-queue
```

## Quickstart

```js
import { createQueue } from "relay-queue";

const queue = createQueue({
  concurrency: 2,
  retries: 3,
  retryDelayMs: 250,   // then 500ms, then 1s
  timeoutMs: 10_000,   // per attempt
  onError: (err, jobId) => console.error(`job ${jobId} failed`, err),
});

queue.add(() => sendEmail("welcome", user));
queue.add(() => chargeCard(order), { priority: 10, timeoutMs: 5_000 });

await queue.onIdle(); // resolves when everything is done
queue.stats(); // { added, completed, failed, retried, timedOut, running, pending }
```

## Concepts

- **Concurrency** — at most `concurrency` jobs run at once; the rest wait.
- **Priorities** — higher `priority` runs first; FIFO within a level, so
  equal-priority work keeps its submission order.
- **Timeouts** — `timeoutMs` bounds each *attempt*; a timed-out attempt
  raises `TimeoutError`, counts as a failure, and consumes a retry.
- **Retries with backoff** — a failing job retries up to `retries` times,
  waiting `retryDelayMs` before the first retry and multiplying by
  `backoffFactor` (default `2`) each time. A job waiting on a retry delay
  still counts as running, so `onIdle()` only resolves when work is truly
  finished.
- **Draining** — `pause()` stops new starts, `clear()` drops the backlog,
  and `onIdle()` awaits completion — the building blocks of graceful
  shutdown (see the [recipe](/together#graceful-shutdown)).

## When to use it

Relay Queue fits bounded, in-process background work: fan-out API calls,
send batches, import pipelines. It is deliberately not durable — if jobs
must survive a process crash, put a persistent queue (Redis, SQS, Postgres)
in front and use Relay Queue as the in-process executor.

See the [API reference](/relay-queue/api) for the full surface.
